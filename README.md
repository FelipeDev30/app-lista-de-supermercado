# App Lista de Supermercado

Este é um aplicativo simples de lista de supermercado desenvolvido em HTML, CSS e JavaScript puro.

## Funcionalidades

- Adicione produtos com nome e valor à lista.
- Visualize todos os produtos cadastrados.
- Veja o valor total da lista em tempo real.
- Limpe toda a lista com um clique.

## Como usar

1. Abra o arquivo [`supermercado/index.html`](supermercado/index.html) em seu navegador.
2. Digite o nome do produto e o valor.
3. Clique em **Cadastrar** para adicionar à lista.
4. O total será atualizado automaticamente.
5. Para limpar a lista, clique no botão **Limpar**.
